# Alert Rules

- Spend change > 30%
- Conversions drop > 30%
- CPA increase > 25%
- CTR drop > 20%
- CPC increase > 20%

## Reporting Behavior
The system should send a daily summary regardless of whether any alerts are triggered.

If no campaigns are flagged, the summary should clearly state that monitored accounts appear stable based on trailing 7-day averages.
